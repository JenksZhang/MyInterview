# jk自己的学习方法

## 1. 通过AI快速检索信息
AI会给出很长的大段信息，如果中途遇到很多不理解的地方，递归学习很麻烦，所以需要减少AI每次回答的篇幅，尽量控制不理解的知识出现的频率，遇到不理解的东西时立即发给AI让其回答，然后再仔细品读此次回答，试图理解，可以适当让AI给出一些例子方便理解，当完成一整块知识的学习时，退到最上层，看看是否可以完全的理解。如果不能理解，请继续这个过程。直到完全理解这一块的知识。结束的时候可以让AI画出一个此次学习的流程图（遇到的知识点，自己的误解区），这样可以加深自己的理解，整理出更适合自己的知识点，方便以后复习。

```mermaid
flowchart TD
  Start([开始学习一块知识])
  A[1. 设定学习目标与范围（小且清晰）]
  B[2. 要点阅读：让AI简短回答（控制长度）]
  C{3. 是否完全理解？}
  C_yes --> D[4. 主动举例并操作/练习]
  D --> E[5. 自测（解释给别人/写一段总结）]
  E --> F{6. 是否能完整复述并应用？}
  F_yes --> G[7. 回到上层：大块检查（整体理解）]
  G --> H{8. 整体是否清楚？}
  H_yes --> I[完成：生成学习流程图 + 误解地图（记录）]
  H_no --> B
  C_no --> J[3a. 立即把不懂的点发给AI（精确问题）]
  J --> K[3b. AI 给出短回答 + 例子（只要点）]
  K --> L[3c. 阅读并尝试用例子验证]
  L --> C
  F_no --> J
  Start --> A --> B --> C

  style Start fill:#f9f,stroke:#333,stroke-width:1px
  style I fill:#bfffbf,stroke:#2c7,stroke-width:1px
```
